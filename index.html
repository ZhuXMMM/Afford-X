Abstract

Afford-X explores scalable methods for affordance reasoning in robotics. Leveraging large-scale datasets and advanced reasoning models, it demonstrates improved generalization across diverse tasks, enabling robots to perform complex actions with minimal supervision. This project integrates multi-modal data and domain-specific insights to push the boundaries of robot affordance reasoning.

Highlights

- Utilizes large-scale datasets for task and object generalization.
- Integrates robust reasoning models for diverse affordance scenarios.
- Offers open-source code and comprehensive resources for researchers.
